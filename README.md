# 手話技能検定 学習支援ツール

<img src="./img/IMG_logo.png" width="30%">

手話技能検定の動画問題を、ランダムに繰り返し学習するためのWebアプリケーションです。
## 概要
ユーザーが選択した複数の動画ファイルをシャッフルし、1つの問題につき2回ずつ連続で再生します。
「動画を何度も選び直す手間」を省き、効率的な反復練習をサポートします。

## 主な機能
- **動画のシャッフル再生**: 読み込んだファイルをランダムな順序で再生します。
- **2回連続再生**: 実際の検定での出題形式と同じ様に、1つの問題を2回連続で流します。
- **解答表示制御**: 2回目再生時にのみ解答（ファイル名）を表示するON/OFF機能を搭載。
- **レスポンシブ対応**: スマートフォンでもPCでも学習可能です。

## こだわった設計・リファクタリング
このプロジェクトでは、コードの品質とメンテナンス性を高めるために以下の工夫を行いました。

- **状態管理の集約**: アプリの状態（ファイル、再生順、インデックス等）を1つの `state` オブジェクトにまとめ、データの流れを可視化しました。
- **メモリ管理の最適化**: `URL.createObjectURL` で生成したメモリを、ファイル再読み込み時に `revokeObjectURL` で適切に解放し、ブラウザの負荷を軽減しています。
- **命名規則の遵守**: Airbnb JavaScript Style Guideを参考に、意味の通じる変数名・関数名を採用しました。
- **モダンJSの活用**: `flatMap` を用いた再生リスト生成や、アロー関数によるイベントハンドリングを行っています。

## 使用技術
- HTML5 / CSS3 (Flexbox)
- JavaScript (Vanilla JS / ES6+)
- ESLint (Airbnb-base)
- PlantUML（設計図の作成）

## 使い方
1. スマートフォン等のカメラで撮影するなどして、複数の問題となる動画ファイルを、ファイル名を解答プラス拡張子の形にして任意の場所に保存します。
2. 「問題ファイル選択」ボタンから、学習したい動画ファイルを複数選択します。
3. 自動的にシャッフルされ、再生が始まります。
4. 必要に応じて「解答ON/OFF」を切り替えて答え合わせを行います。
